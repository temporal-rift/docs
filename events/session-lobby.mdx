Events produced by `game-service` session module.

## LobbyCreated 🌐

```json
{ "lobbyId": "uuid", "hostPlayerId": "uuid", "createdAt": "ISO-8601" }
```

## PlayerJoinedLobby 🌐

```json
{ "lobbyId": "uuid", "playerId": "uuid", "playerName": "string" }
```

## PlayerLeftLobby 🌐

```json
{ "lobbyId": "uuid", "playerId": "uuid" }
```

## HostTransferred 🌐

```json
{ "lobbyId": "uuid", "previousHostId": "uuid", "newHostId": "uuid" }
```

Emitted when the host leaves a lobby that still has other players. Always followed by `PlayerLeftLobby` for the previous host in the same transaction.

## LobbyClosed 🌐

```json
{ "lobbyId": "uuid", "gameId": "uuid" }
```

Emitted when the lobby is disbanded — either explicitly via `CloseLobby` or implicitly when the host leaves as the sole remaining player.

## FactionsDrawn 🌐

```json
{ "gameId": "uuid", "lobbyId": "uuid", "factions": ["ERASERS", "PROPHETS", "WEAVERS"] }
```

Lists which factions are in play this game. Does not reveal assignments.

## FactionAssigned 🔒

```json
{ "gameId": "uuid", "playerId": "uuid", "faction": "ERASERS" }
```

Produced once per player. The `notification` module delivers each copy only to the assigned player.

## GameStarted 🌐

```json
{
  "gameId": "uuid",
  "lobbyId": "uuid",
  "playerIds": ["uuid", "uuid", "uuid"],
  "totalFactions": 3,
  "deckSize": 30
}
```

## GameStartFailed 🌐

```json
{ "gameId": "uuid", "lobbyId": "uuid", "reason": "string" }
```

Emitted when faction assignment fails midway. All already-assigned factions are rolled back and the lobby returns to `WAITING`. `reason` is a human-readable failure description (e.g. `"FACTION_ASSIGNMENT_FAILED"`).

## GameStartCancelled 🌐

```json
{ "gameId": "uuid", "lobbyId": "uuid" }
```

Emitted when a player disconnects during game start steps 1–5. The lobby returns to `WAITING`.

## PlayerDisconnected 🌐

```json
{ "gameId": "uuid", "playerId": "uuid" }
```

Emitted when a player's WebSocket disconnects during an active game. Notifies all remaining players. A reconnect grace timer starts immediately — see [PlayerReconnectSaga](/sagas/special#playerreconnectsaga).

## PlayerAbandoned 🌐

```json
{ "gameId": "uuid", "playerId": "uuid" }
```

Emitted when the reconnect grace timer expires. The player is auto-skipped for remaining rounds. If this was the last remaining player, `GameEndSaga` is triggered.

## EraStarted 🌐

```json
{ "gameId": "uuid", "eraNumber": 1, "cascadedEventIds": ["uuid"] }
```

`cascadedEventIds` is empty on era 1. On subsequent eras it lists events carried forward from paradox cascades.

## EventsDrawn 🌐

```json
{
  "gameId": "uuid",
  "eraNumber": 1,
  "events": [{
    "eventId": "uuid",
    "title": "string",
    "outcomes": [
      { "outcomeId": "uuid", "description": "string", "initialProbability": 33 },
      { "outcomeId": "uuid", "description": "string", "initialProbability": 33 },
      { "outcomeId": "uuid", "description": "string", "initialProbability": 34 }
    ],
    "isCascaded": false
  }]
}
```

Cascaded events carry their last known probability state — not reset to 33/33/34.

## HandDealt 🔒

```json
{
  "gameId": "uuid",
  "eraNumber": 1,
  "playerId": "uuid",
  "cards": [
    { "cardInstanceId": "uuid", "cardType": "PUSH" },
    { "cardInstanceId": "uuid", "cardType": "SCAN" },
    { "cardInstanceId": "uuid", "cardType": "JAM" },
    { "cardInstanceId": "uuid", "cardType": "COLLIDE" },
    { "cardInstanceId": "uuid", "cardType": "AMPLIFY" }
  ]
}
```

`cardInstanceId` distinguishes two copies of the same card type in the same hand.
