All players share the same card pool. **5 cards dealt per player** at the start of each era. Deck has **30 cards total**.

## Probability Shifters

| Card | Effect | `cardType` |
|---|---|---|
| **Push** | +20% to one outcome | `PUSH` |
| **Suppress** | −20% to one outcome | `SUPPRESS` |
| **Swing** | Move 30% from one outcome to another on same event | `SWING` |
| **Amplify** | Double effect of next card played this round | `AMPLIFY` |

## Information

| Card | Effect | `cardType` |
|---|---|---|
| **Intercept** | See one card from another player's hand secretly | `INTERCEPT` |
| **Scan** | See current probability state of one event (normally hidden) | `SCAN` |
| **Trace** | See which player influenced a specific event last round | `TRACE` |
| **Decoy** | Play a fake action visible to others but with no effect | `DECOY` |

## Disruption

| Card | Effect | `cardType` |
|---|---|---|
| **Jam** | Target player cannot use special actions next round | `JAM` |
| **Stall** | Delay resolution of one event by one round | `STALL` |
| **Redirect** | Move another player's last probability shift to a different outcome | `REDIRECT` |
| **Nullify** | Cancel the last card played by any player this round | `NULLIFY` |

## Paradox

| Card | Effect | `cardType` |
|---|---|---|
| **Collide** | Force two outcomes to equal probability, guaranteeing a paradox | `COLLIDE` |
| **Stabilize** | Prevent a paradox from triggering on one event this round | `STABILIZE` |
| **Detonate** | If a paradox exists, double its negative score effect on all players except you | `DETONATE` |

## Card categories

Cards are grouped by category in the domain model. The `CardType` enum carries a `getCategory()` method, and `CardType.byCategory(CardCategory)` returns the full set of types in a category.

```
PROBABILITY_SHIFTER  →  PUSH, SUPPRESS, SWING, AMPLIFY
INFORMATION          →  INTERCEPT, SCAN, TRACE, DECOY
DISRUPTION           →  JAM, STALL, REDIRECT, NULLIFY
PARADOX              →  COLLIDE, STABILIZE, DETONATE
```

The `RoundSummaryPublished` event exposes only category and family — never the specific card — to preserve information asymmetry for all players.
